# 🛒 VatanBazaar

## 📌 Overview

VatanBazaar is an e-commerce platform currently under development, aimed at providing a scalable and efficient online shopping solution.

The project is being built with a strong backend foundation using **Node.js and MongoDB**, starting with a secure authentication system.

> 🚧 This project is actively under development.

---

## 🚀 Current Features

* User authentication system (Login)
* Backend API structure setup
* Database integration with MongoDB
